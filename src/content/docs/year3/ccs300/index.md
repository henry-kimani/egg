You will learn about physical security which entails protecting people, property
and assets using physical means such as physical barriers.

Technical security involves using technology based systems to detect and monitor
any threats. 

Both physical and technical security work together to help organizations protect
their assets.

You will also learn how to assess a facility for threats and vulnerabilities,
and mitigate them.
