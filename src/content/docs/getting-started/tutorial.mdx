## Updates | Release Notes

### What are they?

Release notes are a user friendly preview of the updates made, features added 
and bugs fixed. They are ment to keep you up to date with recent changes made 
in Egg.

We released them for every major or minor update of Egg.

### Where do I find them?

Release Notes are currently divided into two.

##### Quick preview

To quickly skim through the Release Notes. You can find a <a href="/getting-started/new">list of the release
notes</a> here.

##### Detail preview

To read a more indepth changes made in Egg for a specific version.

To access a detailed preview of a
Release Note, you will have to search it in the <a href="getting-started/new">
list of release notes</a> and click on heading. (For that version)

If you are interested in <a href="/changelog">Egg's changelog</a>, get it here.



## Theme

Since we care about your eyes and all the "All Nighters", we have two themes,
light, to use it in bright environments and dark low lit environments.

By default, the theme is set to System Default (auto).

To change the theme for PC, navigate to the Top Bar on the far right.

For Mobile:
